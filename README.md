# Blog Website

---

###  Description: 
Full stack web application that allow users to register, login, create/ edit /delete post. Frontend with React backend with mongo/express

### Tech Stack: 
     - MongoDB, 
     - Express, 
     - React 
     - Node.  
     - HTML, 
     - CSS
